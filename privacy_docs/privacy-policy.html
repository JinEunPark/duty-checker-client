개인정보 처리방침

시행일: 2026년 4월 12일

"오늘안부"(이하 "앱")는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보 처리방침은 앱이 수집하는 개인정보의 항목, 이용 목적, 보관 기간 및 이용자의 권리에 대해 설명합니다.

1. 수집하는 개인정보 항목

항목 수집 목적 수집 방법
전화번호 회원 식별, 로그인, 보호자/당사자 연결 사용자 입력 (SMS 인증)
비밀번호 로그인 인증 사용자 입력 (암호화 저장)
사용자 역할 (당사자/보호자) 서비스 기능 분기 회원가입 시 선택
디바이스 토큰 (FCM) 푸시 알림 전송 앱 설치 시 자동 생성
안부 체크인 기록 안부 확인 상태 제공 사용자 액션 (체크인 버튼)
보호자/당사자 연결 정보 안부 확인 관계 관리 사용자 입력 (전화번호)

2. 개인정보의 이용 목적

  • 회원 가입 및 본인 인증
  • 당사자와 보호자 간 안부 확인 서비스 제공
  • 안부 미확인 시 보호자에게 푸시 알림 전송
  • 서비스 이용 관련 공지 및 안내

3. 개인정보의 보관 및 파기

  • 회원 탈퇴 시 모든 개인정보를 즉시 파기합니다.
  • 관련 법령에 의해 보존이 필요한 경우, 해당 기간 동안 별도 보관 후 파기합니다.
보관 항목 보관 기간 근거
회원 정보 (전화번호, 역할) 회원 탈퇴 시까지 서비스 제공
안부 체크인 기록 회원 탈퇴 시까지 서비스 제공
서비스 이용 기록 3개월 통신비밀보호법

4. 제3자 제공

앱은 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우

5. 외부 서비스 이용

서비스 제공자 목적 처리 항목
Firebase Cloud Messaging Google LLC 푸시 알림 전송 디바이스 토큰

6. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람, 정정, 삭제 요청
  • 개인정보 처리 정지 요청
  • 회원 탈퇴 (앱 내 설정 화면에서 가능)

7. 개인정보의 안전성 확보 조치

  • 비밀번호 암호화 저장
  • HTTPS를 통한 데이터 전송 암호화
  • JWT 토큰 기반 인증 및 자동 갱신
  • 접근 권한 최소화

8. 개인정보 보호책임자

9. 방침 변경

본 개인정보 처리방침이 변경되는 경우, 앱 내 공지 또는 푸시 알림을 통해 사전에 안내합니다.
